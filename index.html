Programming

Use ST-Link Utility

Read Out Protection set Disable

Open file :

F5V100_Beeper_ON_110C.bin

Program & Verify

Connect to OBDII (for test only)

 

Only 3 wire use. Black GND, Red +12V, Brown-red Can-H (GMLAN). Wires GND and GMLAN must be twisted.

Warning: Idle current consumption = 55mA.

Get +12V from pin AR from CD30MP3 or lamp glove box / lamp power outlet (cigarette lighter).
